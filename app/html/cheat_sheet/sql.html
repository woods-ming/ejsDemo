Struct Query Language

摘要: sql是结构化查询语言，用于访问关系型数据库

关键字: sql，结构化查询语言

一、数据定义

1.Table 表

创建
create table 【表】(
  【字段】 【数据类型】 【约束】,
  ...
)
修改
alter table 【表】 add 【字段】 【数据类型】 【约束】
alter table 【表】 modify 【字段】 【数据类型】 【约束】
alter table 【表】 drop column 【字段】
alter table 【表】 rename column 【字段】 to 【新字段名】
rename 【表】 to 【新表名】
删除
drop table 【表】
查看表结构
desc 【表】

2.Sequence 序号

创建
create sequence 【序号】
  minvalue 【值】
  maxvalue 【值】
  start with 【值】
  increment by 【值】
  cache 【值】
修改
alter sequence 【序号】 【属性】 【值】
删除
drop sequence 【序号】

修改

二、数据操作

1.Tables

插入
insert into 【表】(【各字段的名称】) values(【一条记录】)
insert into 【表】(【各字段的名称】) (【记录集合】)
更新
update 【表】 set 【字段】=【值】 where 【条件】
删除
delete from 【表】 where 【条件】
truncate table 【表】（删除全部数据，速度快）

三、数据查询

1.Select

单表查询
select 【字段】 from 【表】
  where 【条件】
  group by 【列名】
  order by 【列名】 asc/desc
modify
delete
