Windows Azure发布网站

摘要: 了解如何使用微软的云服务器Windows Azure发布网站

关键字: Windows Azure，发布网站

一、在云服务器的门户中操作

1.登录manage.windowsazure.cn

1.初始化数据库

2.部署网站

提取发布设置信息
host：用“publishUrl="ftp://”查找，记住“/site/wwwroot”之前的部分
userName：用“userName”查找，记住对应的值
userPWD：用“userPWD”查找，记住对应的值
安装FileZilla
https://filezilla-project.org/

主机：粘贴刚才找到的host
登录类型：正常
用户：粘贴刚才找到的userName
密码：粘贴刚才找到的userPWD
