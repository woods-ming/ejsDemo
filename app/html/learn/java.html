初学Java

摘要: 从零开始学习Java

关键字: Java入门

一、搭建开发环境

1.安装Jdk

包含内容
JRE、编译器、打包工具、文档生成器、调试器、头文件生成器、反汇编器、监控工具等
下载
http://www.oracle.com/technetwork/java/javase/downloads/index.html
JAVA_HOME
C:\Program Files\Java\jdk1.7.0
CLASSPATH
.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
Path
%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
指定java版本
java -version:

2.安装Tomcat

下载（Core:zip）
http://tomcat.apache.org/download-80.cgi
绿色
解压到不需要Admin权限的目录
CATALINA_HOME
F:\Downloads\Tomcat\apache-tomcat-8.0.22
启动
双击 ‘安装目录\bin\startup.bat’ || 打开浏览器 ‘http://localhost:8080/’

3.安装Eclipse

下载
http://www.eclipse.org/downloads/
绿色
解压到不需要Admin权限的目录
工作目录
存放代码的默认文件夹
配置运行环境
Preference\Java\Installed JREs：选择jdk安装目录
配置Tomcat
Preference\Server\Runtime Enviorments：添加Apache Tomcat v8.0及安装目录 || JRE选择jdk开头的
配置Content Assist
Preference\Java\Content Assist\Auto activacation：输入'.abc...z'
打开Tomcat
Window\Show View\Servers：右键 start
配置编码格式
General\WorkSpace\Text file encoding：UTF-8
Web\JSP Files\Encoding：ISO 10646/Unicode(UTF-8)
关闭拼写检查
General\Editors\Text Editors\Spelling：禁用拼写检查
安装插件
Help\Eclipse Marketplace

4.安装Oracle

下载（11g）
http://www.oracle.com/technetwork/database/enterprise-edition/downloads/index.html 两个文件都下载
安装
两个文件解压到同一个目录，双击 setup.exe
删除
安装目录\product\11.2.0\dbhome_1\deinstall， 双击 deinstall.bat
ORACLE_HOME
F:\oracle\product\10.2.0\db_1

5.安装PL/Sql Developer

下载（破解版）
http://gd.ddooo.com:8081/uuauth/PLSQLDeveloper_13356.rar
配置
Tools\Preferences ...\Connection, Oracle Home使用自动检测

二、Jsp（方便生成Html的servlet，由Tomcat自动解析）

1.Hello World

创建项目
Web\Dynamic Web Project
创建页面
WebContent\某目录，右键 JSP File
编写代码
跟Asp差不多，语句用<% %>，声明用<%! %>，表达式用<%= %>
九大内置对象
request，response，session，application，out（输出），page，pageContext，config，exception
部署到Tomcat
Server\Add and Remove...，添加当前项目
运行
Window\Show View\Servers：右键 start || 打开浏览器 ‘http://localhost:8080/目录/xxx.jsp’
（修改端口号）
并列工程目录的Servers\server.xml：找到Connector\port，进行修改
（智能提示）
快捷键：Alt + /
导入运行时类库：项目右键 Build Path\Config Build Path\Java Build Path\Add Library\Server Runtime
（解决中文乱码）
post：request.setCharacterEncoding("utf-8");
get：修改Tomcat的server.xml的配置 URIEncoding="utf-8"

2.使用JavaBean

四大特点
public， 无参构造函数， 私有属性， get/set访问器（符合特定规范的java类，用于做Model层）
<jsp:useBean />
实例化Bean，并将Bean作为成员，挂到scope对象上（它就具有了scope对象的生存期）
<jsp:setProperty />
给指定Bean的属性赋值（从表单或Request的池子中捞数据，与.net MVC中的模型绑定极其相似）
<jsp:getProperty />
读取指定Bean的属性值
本质
jsp要经常使用Bean，因此特定开发出一系列语法技巧

3.使用Jsp指令

<%@ include file %>
嵌入外部文件的原始代码（物理替换）
<%@ page import %>
相当于 c#的using
<%@ page contentType %>
当前页面使用的字符编码（推荐utf-8）
本质
编译Jsp页面时，执行的逻辑

4.使用Jsp动作

<jsp:include page />
嵌入外部文件的执行结果（逻辑调用）
<jsp:forward page />
服务器跳转（交给指定Jsp页面处理请求）
<jsp:param name value />
作为forward动作的子标签，给request对象添加请求参数（相当于 c#的routeValues）
本质
运行Jsp页面时，调用的逻辑

四、Oracle基础

1.系统用户

概念
系统自带的4个用户
sys
sysdba/sysoper
system
normal
sysman
normal
scott
tiger
登录
connect 用户名/密码 @服务器地址 as normal/sysdba/sysoper
查看登录用户
show user
用户的数据字典
dba_users
启用用户
alter user 用户名 account unlock;

2.表空间

概念
数据的逻辑分类，包含多个数据文件
永久表空间
表、视图、存储过程等
临时表空间
执行时的中间过程，执行完删除
undo表空间
为事务服务，保存修改前的数据
dba_tablespaces / user_tablespaces
表空间相关的信息
dba_data_files / user_data_files
数据文件相关的信息
修改默认表空间
alter user 用户名 default tablespace 表空间
创建表空间
create tablespace 表空间 datafile 'xxx.dbf' size 10m;
删除表空间
drop tablespace 表空间 including contents;
增加数据文件
alter tablespace 表空间 add datafile 'xxx.dbf' size 10m;
删除数据文件
alter tablespace 表空间 drop datafile 'xxx.dbf';

3.表

概念
二维结构的基本存储单位
数据类型
字符型 char(n)/nchar(n), varchar2(n)/nvarchar2(n)
数值型 number(有效数字,小数位数), float(n)
日期型 date, timestamp
大对象 blob【二进制存储】, clob【字符串存储】
约束
非空约束 not null
主键约束 primary key / constraint 【约束名】 primary key(【字段1,2...】)
外键约束 reference 【主表】(【字段】) / constraint 【约束名】 foreign key(【从表字段】) references 【主表】(【主表字段】) on delete cascade
唯一约束 unique / constraint 【约束名】 unique(【字段】)
检查约束 check(【条件】) / constraint 【约束名】 check(【条件】)
约束的数据字典
